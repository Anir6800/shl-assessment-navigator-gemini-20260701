SHL Assessment Navigator

Grounded assessment recommendations without catalog hallucinations.

This interface talks to the stateless FastAPI service in the project. It clarifies vague asks, recommends SHL assessments, refines shortlists, and compares catalog items using only catalog data.

Stateless Every call sends the transcript again, so the backend never stores conversation memory.
Grounded Recommendations come from the SHL catalog JSON endpoint and cached vector index.
Dockerized Ready for container deployment with Gemini, Anthropic, or OpenRouter through env vars.
PDF report The implementation report is generated as a polished PDF under the outputs folder.
